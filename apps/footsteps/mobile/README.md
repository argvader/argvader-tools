## Footsteps Mobile Application ##
React Native application written in cljs.
Creates fun sounds when you walk :)

## Build ##
Using shadow-cljs for hot reload and Expo for emulation


Two parts right now:
 > clj -R:1.10.1 -A:shadow-cljs watch app

and yarn start
