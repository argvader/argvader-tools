Go blocks provide threadlike concurrency for JavaScript

          
(go (while true
  (<! (timeout 250)) 
  (>! c "Red")))
Go 
          
(go (while true
  (<! (timeout 1000)) 
  (>! c "Blue")))
Go 
          
(go (while true
  (<! (timeout 2000))
  (>! c "Green")))
